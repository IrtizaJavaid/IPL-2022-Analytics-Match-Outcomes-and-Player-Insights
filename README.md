# This project presents an exploratory data analysis of the Indian Premier League (IPL) 2022 season. Using Python libraries such as Pandas, Seaborn, Matplotlib, and Plotly, the analysis highlights team performances, toss outcomes, player contributions, and venue statistics.
Key Insights
## Team Performance
--Most match wins by team
--Toss decision trends (bat vs field)
--Toss winner vs match winner
--How teams win (Runs vs Wickets)
## Player Highlights
--Most “Player of the Match” awards
--Top scorers (frequency and cumulative runs)
--Highest individual score
--Best bowling figures
## Venue Analysis
--Venues hosting the most matches

## Tools & Libraries
--Python: Pandas, NumPy
--Visualization: Seaborn, Matplotlib, Plotly
